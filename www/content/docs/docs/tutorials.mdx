# Tutorials

The following tutorials cover setting up Gatsby Theme Catalyst based websites in more detail.

[Building a website for a freelance writer](/docs/tutorials/building-a-website-for-a-freelance-writer-using-gatsby-theme-catalyst-hydrogen/): Provides a detailed walkthrough of setting up `gatsby-theme-catalyst-hydrogen` for as a personal portfolio for a freelance writer.

[Creating a blog](/docs/tutorials/creating-a-blog-with-gatsby-theme-catalyst-lithium/): Instructions to setup a personal blog using `gatsby-theme-catalyst-lithium`. This tutorial would also be relevant for `gatsby-theme-catalyst-helium` which is a very similar blogging theme.

[Setting up Gatsby Theme Catalyst Sanity](/docs/tutorials/setting-up-gatsby-theme-catalyst-sanity/): Instructions to setup a barebones implementation of `gatsby-theme-catalyst-sanity` which can be used as a base to build a completely custom site. For an example of a complete site based on this theme take a look at how it is used in `gatsby-theme-catalyst-bery`.
